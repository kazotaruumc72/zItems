# zItems Documentation

Welcome to the **zItems** official documentation! zItems is a powerful Minecraft plugin (Paper/Spigot 1.21+) that provides a flexible framework for creating custom items with advanced effects, custom block tracking, and seamless integration with popular plugins.

## What is zItems?

zItems allows you to create highly customizable items with:
- **Advanced Effect System**: Apply powerful effects like Hammer (3D mining), Vein Mining, Auto-Sell, and more
- **Custom Block Support**: Integration with ItemsAdder, Nexo, Oraxen for custom blocks
- **Dynamic Lore Management**: Automatic effect display with full visibility control
- **Extensible API**: Create your own effects and integrate with other plugins
- **Hook System**: Built-in support for Jobs, WorldGuard, Shop plugins, and more

## Features at a Glance

| Feature | Description |
|---------|-------------|
| **14+ Built-in Effects** | Hammer, Vein Mining, Auto-Sell, Silk Spawners, XP Boost, and more |
| **Effect Representation** | Apply effects via Smithing Table or custom GUIs |
| **Custom Block Support** | Works with ItemsAdder, Nexo, Oraxen custom blocks |
| **Dynamic Lore** | Automatic effect display with customizable visibility |
| **Hook System** | Integrates with Jobs, WorldGuard, shop plugins |
| **Event-Driven** | Extensible architecture for custom behaviors |
| **PDC Storage** | Efficient data storage using Persistent Data Containers |
| **Folia Support** | Compatible with Folia multi-threaded server software |
